# One Shots

Un **One Shot** es una partida de juego de rol diseñada para completarse en una sola sesión, generalmente entre 3 a 6 horas. A diferencia de las campañas largas que pueden durar meses o años, los one shots ofrecen una historia completa y autocontenida que se resuelve en una sola tarde o noche de juego.

## Características de un One Shot

### Duración y Estructura
- **Tiempo:** Entre 3 a 6 horas de juego efectivo
- **Historia completa:** Introducción, desarrollo y conclusión en una sola sesión
- **Autocontenido:** No requiere conocimiento previo de campañas anteriores
- **Enfoque directo:** Menos exploración libre, más narrativa dirigida

### Requerimientos del Director

El **Director de Juego (DM/GM)** o creador del one shot debe definir claramente:

#### Nivel de Personajes
- **Nivel específico:** Los personajes deben estar en un rango determinado (ej: nivel 3-5)
- **Justificación:** El nivel debe corresponder con los desafíos y enemigos planeados
- **Experiencia:** Algunos one shots están diseñados específicamente para jugadores nuevos (nivel 1) o veteranos (niveles altos)

#### Tamaño del Grupo
- **Número de jugadores:** Generalmente entre 3 a 6 jugadores
- **Roles sugeridos:** Puede incluir recomendaciones de clases o arquetipos necesarios
- **Escalamiento:** El director debe poder ajustar encuentros según el número final de participantes

#### Sistema de Juego
- **Reglas específicas:** Qué edición o sistema se utilizará
- **Reglas de casa:** Cualquier modificación especial para la sesión
- **Material permitido:** Qué libros o suplementos están disponibles para la creación de personajes

### Uso de Personajes Existentes

Cuando un jugador desea usar un personaje de otra campaña, es esencial llegar a **acuerdos previos con el Director**:

#### Negociaciones Necesarias
- **Ajustes de nivel:** Subir o bajar el nivel del personaje para que coincida con los requerimientos
- **Equipo apropiado:** Modificar el inventario para que sea apropiado para la aventura
- **Trasfondo temporal:** Explicar cómo el personaje llegó a esta situación específica
- **Limitaciones narrativas:** El director puede restringir ciertas habilidades o equipo que rompan la aventura

#### Alternativas de Personajes
- **Personajes pregenerados:** Muchos one shots incluyen personajes listos para usar
- **Creación rápida:** Sistema simplificado para crear personajes nuevos
- **Arquetipos:** Personajes genéricos que los jugadores pueden personalizar rápidamente

## Ventajas de los One Shots

### Para Jugadores
- **Compromiso mínimo:** Ideal para horarios ocupados
- **Experimentación:** Probar nuevas clases, razas o sistemas
- **Punto de entrada:** Perfecto para jugadores nuevos
- **Variedad:** Experimentar diferentes géneros y estilos de juego

### Para Directores
- **Preparación enfocada:** Menos contenido que preparar
- **Experimentación:** Probar nuevas mecánicas o estilos narrativos
- **Prueba de material:** Testear ideas antes de usarlas en campañas largas
- **Flexibilidad de horario:** Más fácil de coordinar con múltiples personas

## Consideraciones Importantes

### Preparación Previa
- **Sesión 0 condensada:** Breve explicación de reglas y expectativas
- **Creación de personajes:** Planificar tiempo extra si los jugadores crean personajes nuevos
- **Materiales:** Asegurar que todos tengan acceso a dados, hojas de personaje y reglas básicas

### Durante la Sesión
- **Gestión del tiempo:** El director debe mantener el ritmo para completar la historia
- **Flexibilidad narrativa:** Estar preparado para acelerar o expandir secciones según sea necesario
- **Enfoque en la diversión:** Priorizar el entretenimiento sobre la perfección mecánica

### Expectativas Realistas
- **Historia condensada:** No esperar la profundidad de desarrollo de una campaña larga
- **Personajes funcionales:** Los personajes pueden ser menos complejos narrativamente
- **Resolución rápida:** Algunos hilos narrativos pueden no explorarse completamente

!!!info Consejo para Directores
Prepara más contenido del que crees que vas a usar, pero estate dispuesto a omitir partes si el tiempo se agota. Es mejor terminar la historia de manera satisfactoria que dejarla incompleta.
!!!

!!!warning Importante
Comunica claramente las expectativas y requerimientos antes de la sesión. Los one shots funcionan mejor cuando todos los participantes entienden el formato y limitaciones desde el inicio.
!!!

Los one shots son una excelente manera de disfrutar los juegos de rol sin el compromiso a largo plazo de una campaña tradicional, ofreciendo experiencias completas y memorables en una sola sesión de juego.
